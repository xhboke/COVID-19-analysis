# COVID-19-analysis

基于传染病SIR模型对世界新冠病毒的分析、包含死亡率，治愈率等，以及世界主要国家疫情情况展示

## SIR模型

[https://baike.baidu.com/item/SIR%E6%A8%A1%E5%9E%8B/1938321](https://baike.baidu.com/item/SIR%E6%A8%A1%E5%9E%8B/1938321)

2020-01-22 -> 2021-04-30
